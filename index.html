Summary of Weather Data


The purpose of this dashboard is to summarize a set of weather data and its corresponding plots. The dashboard should clearly and be easily disseminated by the user. Presenting the weather data via a web site allows the user easy access to data for review and analysis.

All data was collected through multiple APIs to collect geolocation information for cities being surveyed. This project only surveyed the following data: Maximum temperature, humidity, cloudiness, and wind speed. Weather data was cleaned and reduced using Python and then summarized by using the Python package Matplotlib.
